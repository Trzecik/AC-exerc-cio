## Exercício
>Fórmula

Número de bits por segundo = largura do barramento (bits) * velocidade de operação (Hertz)


---

> Calculo Mb/s

 64 * 1200 = 76 800 Mb/s

---
> Calculo bits/s

 64 * 1200 = 76 800 Mb/s

<=> 76 800 * 8 000 000 = 614 400 000 000 bits/s

---


 > Calculo GHz/s

 64 * 1.2 =  76.8 GHz/s

---
 ### Tabela com Resultados
 
 | Unidades | Conversão |
| ----------- | ----------- |
| Bits/s | 614 400 000 000  |
| Mb/s | 76 800 |
| GHz/s | 76.8 |
